API mini project: URL Shortner

  • Pass a URL with preceeding /new/. If the URL is valid, a JSON object containing the shortened URL is returned.

  • Visit the shortened URL to be redirected to the original URL

    • Example insert usage:

    https://sanggone-shorturl.herokuapp.com/new/http://www.google.ca
    https://sanggone-shorturl.herokuapp.com/new/https://www.amazon.ca

    Example insert output:

    {"original_url":"http://www.google.ca","short_url":"https://sanggone-shorturl.herokuapp.com/5"}
    {"original_url":"https://www.amazon.ca","short_url":"https://sanggone-shorturl.herokuapp.com/3"}

    Example usage:

    https://sanggone-shorturl.herokuapp.com/3

    Redirected to:

    https://www.amazon.ca
